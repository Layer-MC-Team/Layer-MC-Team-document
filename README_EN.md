    ✨ Project Introduction
This repository is a complete collection of technical documents for Minecraft: Java Edition launchers on Android. It includes in-depth notes about mobile launcher architecture, graphics translation layers, Vulkan drivers, JVM configuration, mod compatibility, crash troubleshooting and more.

All documents are provided with original Chinese text and translated English versions, for MC launcher developers and technical enthusiasts to learn the logic & fixes of running Minecraft Java on mobile devices.

    🚀 Core Content
Architecture analysis of mainstream Android MC launchers: PojavLauncher / FoldCraftLauncher / Mobile HMCL
Graphics translation tech: GL4ES, Zink, VirGL, ANGLE and adaptation guides
Multi-version JVM setup, memory tuning & custom Java import tutorials
Mobile compatibility records for mod loaders: Forge / NeoForge / Fabric / Quilt / OptiFine
Crash & exception (XXXException) analysis & real-world fixes
